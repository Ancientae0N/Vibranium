# Vibranium

This project implements the frontend component of this project.

## Installing

1. Install `node` and `npm`
2. From the root, `cd` to the `vibranium filder.
3. Install the dependencies:
```js
npm install
```
4. Install the vue cli:
```js
npm install -g vue-cli
```
5. Start the server:
```js
cd src
vue serve
```
